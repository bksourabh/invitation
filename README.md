# Mazumder Family Invitation

A family invitation website for Australian immigration purposes, hosting a formal invitation letter for Biman Kanti Mazumder and Bandana Mazumder's visit to Melbourne, Australia.

## Hosting

This site is designed to be hosted on **GitHub Pages**. To deploy:

1. Go to your repository's **Settings** > **Pages**
2. Under **Source**, select the branch (e.g., `main`) and root (`/`)
3. Click **Save** — the site will be live at `https://<username>.github.io/Invitation/`

## Files

- `index.html` — Main website
- `styles.css` — Stylesheet
- `script.js` — Scroll animations
- `Invitation_Letter_Biman_Bandana_Mazumder_Australia_Visit.pdf` — Downloadable formal invitation letter
- `letter-pdf.html` — Source HTML used to generate the PDF
